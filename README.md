# DemoQA Automation Framework

Python automation framework for UI and API testing using PyTest, Selenium, and Allure.

## Tech Stack
- Python
- PyTest
- Selenium
- Requests
- Allure
- GitHub

## How to Run

### Run all tests
bash run_allure.sh

### Run only UI tests
bash run_ui.sh

### Run only API tests
bash run_api.sh

### Run failed tests only
pytest --lf

## Features
- UI & API automation
- Page Object Model
- Explicit waits
- Screenshot on failure
- Allure reporting
- Flaky test handling
